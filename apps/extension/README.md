# Browser Recall Extension

> Chrome extension for local-first semantic search of browser history, powered by Chrome Built-in AI.

## Development

Run from repository root:
```bash
# Development server with hot reload
pnpm --filter @browser-recall/extension dev

# Production build
pnpm --filter @browser-recall/extension build

# Create distribution zip
pnpm --filter @browser-recall/extension zip
```

Or run from this directory:
```bash
pnpm dev
pnpm build
pnpm zip
```
